# -Projeto-L-gico-de-Banco-de-Dados-Oficina-Mec-nica
Desafio de projeto – Módulo de Banco de Dados Relacionais Bootcamp DIO | Engenharia de Dados
Este repositório contém o Projeto Lógico de Banco de Dados para o contexto de uma oficina mecânica, desenvolvido a partir do esquema conceitual (modelo ER) criado no módulo anterior. O projeto contempla:

Esquema lógico relacional
Script DDL de criação das tabelas
Script DML de inserção de dados para testes
Conjunto de queries SQL respondendo perguntas de negócio
